
# js基础项目

该项目主要用于

1. 当我们需要做写一些简单的小dojo的时候。
2. 当我们要写几行代码看看效果的时候.


# 使用方法

准备环境：

    npm install

执行测试：

    npm test
    
执行index.js
    
    npm run-script index
